Git Cheat Sheet

by villed3
Init
Initialize local repository in current directory: 
git init
Clone
Clone repository into subdirectory of current directory: 
# remote
git clone <server>

# local
git clone /path/to/local/repository
Status
Get status of repository, showing which modified files that are staged/unstaged/untracked and whether branch is up to date with remote: 
git status
Branches
Create new branch: 
git checkout -b <branch_name>
Switch to another branch: 
git checkout master
git checkout <branch_name>
Merge
Merge branch into main or other branch: 
git merge main
git merge <branch>

Conflicts must be solved if merge does not succeed. Edit affected files and add them again. It can be easier to merge using the graphical interface of your git server website.

Discard
Reset an unstaged file to its last commited version: 
git checkout -- <filename>
Add & Commit
Add modified files to the index (stage them): 
# single file
git add <filename>

# entire subdirectory
git add some/directory/

# all files
git add *
Commit the staged files and provide a message: 
git commit -m "A short but descriptive message"
Pull
Update your local repository branch to latest from active remote: 
git pull
Push
Upload a branch to the remote location: 
git push origin master
git push <remote> <branch>
Remote
Add a remote location to push from, with a custom alias: 
git remote add <alias> <server>
Verify that remotes are working: 
git remote -v
Config
View current configured options for the local repository: 
git config --list
Modify the value of an option: 
git config <option> <value>

If you use different git servers using different account details, you will need to change user.name and user.email for each local repository.
